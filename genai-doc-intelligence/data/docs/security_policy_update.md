# Security Policy Update - Remote Access Standard

Effective date: January 10, 2026
Owner: Security and Infrastructure Team

Policy requirements:
- All remote access to production systems requires company-managed devices.
- Multi-factor authentication is mandatory for VPN, cloud consoles, and privileged admin tools.
- Production database exports must be approved by the data governance lead and logged in the audit register.
- Credentials may not be shared through chat, email, or ticket comments.
- Critical security patches must be applied within seven calendar days of vendor release.

Exceptions:
- Temporary exceptions require written approval from the CISO and expire after thirty days unless renewed.

Enforcement:
- Violations may result in suspension of system access, formal disciplinary action, or both.
